# Terraform

### Full deploy

Install project dependencies

~~~bash
terraform init
~~~

#### terraform deployment

~~~bash
terraform apply
~~~

#### terraform deployment details

Lets you output a terraform plan with all the details you might be familiar with.

~~~bash
terraform init
terraform plan -no-color
terraform apply
terraform destroy
~~~
